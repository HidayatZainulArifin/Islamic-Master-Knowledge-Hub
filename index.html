🚀 Master Reference: Sifat Nabi Muhammad SAW

Penjelasan ringkas tentang 3 Kategori Sifat yang dimiliki para Nabi:

/* 1. Sifat Wajib (Must Have) */
Nabi.sifatWajib = {
    siddiq: "Selalu Benar/Jujur",
    amanah: "Sangat Terpercaya",
    tabligh: "Menyampaikan Wahyu",
    fathonah: "Sangat Cerdas"
};

/* 2. Sifat Mustahil (Impossible) */
Nabi.sifatMustahil = {
    kizib: "Berbohong",
    khianat: "Berkhianat",
    kitman: "Menyembunyikan",
    baladah: "Bodoh"
};

/* 3. Sifat Jaiz (Humanly) */
Nabi.sifatJaiz = "Aradhul Basyariyah (Sifat Kemanusiaan Biasa)";
// Contoh: Makan, Minum, Sakit, Tidur, dan Menikah.
